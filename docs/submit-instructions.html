Description (100pts)

Complete your Java FX program that solves this problem: Program Design: Problem Statement -- Food Query and Meal Analysis

Submission: submit only food.zip

NOTE: you must create your executable.jar and food.zip as described here.
CAUTION: food.zip must contain screenshot, executable.jar, and all source files as described in this document.

Practice these steps first and submit, then do again when your program is working.
These instructions assume that your project files are on CS files servers.
Try your Java FX program on a CS Linux Workstation
All source files must be in package application

  1. Copy or create manifest.txt and manifest_srcbin.txt files and add them to your project folder.
    Note: the blank last line is required.

  2. Copy or create this Makefile and add it to your project folder.
     make nosrc             # will build executable when there are not separate src and bin folders for your project
     make srcbin            # will build executable when there are separate src and bin folders for your project

  3. Create an executable.jar using the above Makefile targets 
    Note: you must repeat this step when program code changes
    1. launch a terminal window (if the make utility is not available, you must type commands each time)
    2. navigate to your project folder (the parent folder of your application package folder)
    3. confirm that the above manifest and Makefile files are present in the project folder
    4. If your source (.java) files are in same folder as class files, run:     make nosrc
                      javac -d . application/*.java
                      jar -cfm executable.jar manifest.txt application

    5. Or, if your java files are in separate src and bin folders, run:         make srcbin
                      javac -d bin src/application/*.java
                      jar -cfm executable.jar manifest_srcbin.txt -C bin application

  4. Create a screenshot.jpg showing meal analysis (must repeat this step if layout changes)
    • run your program with provided data file
    • add a food
    • create a meal
    • analyze the meal
    • take a screenshot of the view showing meal and meal analysis
    • save as screenshot.jpg or screenshot.png in your project directory

  5. Create a food.zip file that contains executable.jar, screenshot.png, and all source files.                 make run
    Note: this step must be repeated if anything changes.

    The first target in the Makefile above will run your executable and create a zip file that includes all files inside your project, but does not include your project directory file.  This is what we want.

    • cd project (go to your project folder -- do not go to src folder if you have src folder within project folder)
    • jar -cMf food.zip .

  6. Check that your food.zip file contains the following structure (must repeat if anything changes)

    food.zip for no src and bin folders
    food.zip for src and bin folders

  7. Upload food.zip (this file should be in your project folder - not src folder) (Must repeat this step if anything changes).
